# Dusk UI

A modern React component library built on Base UI, Tailwind CSS v4, and Motion.
Copy-paste components. No black boxes.

## Getting Started

Visit [dusk-ui.vercel.app](https://dusk-ui.vercel.app)

## Tech Stack

- Next.js 16 + React 19
- Base UI 1.0
- Tailwind CSS v4
- tailwind-variants
- Motion v12
- Biome
